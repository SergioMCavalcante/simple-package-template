# calculadora_imc

## Descrição

O pacote calculadora_imc é usado para:

*Calcular o IMC:*

- Calcula o IMC
- Exibe o IMC e a situação da pessoa

## Instalação

Use o gerenciador de pacotes [pip](https://pip.pypa.io/en/stable/) para instalar calculadora_imc

pip install calculadora_imc
